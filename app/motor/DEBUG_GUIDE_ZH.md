## 2. 反电势测试结果换算

已知：

- 线电压 `Vab` 峰峰值：`28.6 V`
- 电角频率：`385 Hz`
- 极对数：`4`

换算公式：

```text
Vll_peak = Vll_pp / 2
lambda = Vll_peak / (sqrt(3) * 2 * pi * fe)
n_mech_rpm = fe / pole_pairs * 60
```

代入结果：

```text
Vll_peak = 28.6 / 2 = 14.3 V
lambda = 14.3 / (1.732 * 2 * pi * 385) = 0.00341 V*s
n_mech_rpm = 385 / 4 * 60 = 5775 rpm
```

所以当前默认参数中：

- `flux_linkage_v_s = 0.00341`
- `max_mech_speed_rpm = 5800`

说明：

- 这里假设你测得的 `385 Hz` 是电角频率，且 `28.6 Vpp` 是 AB 线电压近似正弦波峰峰值。
- 如果后面确认测的是机械频率，或者波形不是正弦线电压，这个磁链值要重新算。

## 3. 电流环 PI 理论公式

当前工程里的电流 PI 已改成按带宽自动计算，用户主要配置的是：

- `Rs`
- `Ld`
- `Lq`
- `current_pi_bandwidth_hz`
- `output_limit`

连续域电流对象可近似写成：

```text
G(s) = 1 / (L*s + R)
```

常用的理论整定公式：

```text
wc = 2 * pi * fc
Kp = L * wc
Ki = R * wc
```

其中：

- `fc` 是电流环设计带宽
- `wc` 是电流环角带宽

当前默认设置：

- `fc = 1000 Hz`
- `Rs = 0.05`
- `Ld = Lq = 0.00045158`

计算结果：

```text
wc = 2 * pi * 1000 = 6283.19 rad/s
Kp_d = Ld * wc = 0.00045158 * 6283.19 = 2.837
Ki_d = Rs * wc = 0.05 * 6283.19 = 314.159
Kp_q = Lq * wc = 0.00045158 * 6283.19 = 2.837
Ki_q = Rs * wc = 0.05 * 6283.19 = 314.159
```

工程里会自动把这组值装载到 `Id/Iq` PI 控制器，不需要你再手填。

## 4. 参数更新原则

- 直接来自硬件原理图和实物测量的内容，放到配置参数里。
- 通过公式能稳定推导出来的内容，尽量不要手填，改成推导参数。
- 电流环优先改带宽，不要一上来直接手改 `kp/ki`。
- `Ld/Lq/Rs/磁链` 这些电机参数优先保证真实，观测器和启动参数再在此基础上调。
- 保护阈值必须和母线量程、采样量程一起看，不能只按经验值填。

## 5. 上电前必须确认的事项

- 三相功率线连接顺序正确，驱动器高低桥逻辑正确。
- 电流采样零点偏置正确，空载静止时两路电流 ADC 值应稳定在中点附近。
- 电流方向极性正确。给定正 `Iq` 时，采样电流和估算转向应一致。
- 母线分压量程确认无误。

当前软件默认调试策略：

- 过流保护开启
- 欠压/过压保护开启
- 过温保护默认关闭
- 缺相保护默认关闭

本板当前 VBus 量程特别要注意：

```text
Vbus_full_scale = Vref * (Rupper + Rlower) / Rlower
                = 5 * (4000 + 1000) / 1000
                = 25 V
```

这意味着：

- ADC 满量程只到 `25 V`
- 过压阈值必须小于 `25 V`
- 如果你后面要跑更高母线电压，必须先改分压网络或改板级参数

## 6. 推荐调试步骤

### 第一步：低压上电，只验证采样

- 先不要带机械负载。
- 先用限流电源、较低母线电压上电，建议第一次从 `12 V` 左右开始。
- 检查串口、示波器或调试器里这几个量：
  - `current_a_raw`
  - `current_b_raw`
  - `vbus_raw`
  - `temp_raw`
- 静止不驱动时，两路电流应接近 ADC 中点，VBus 换算值应接近电源值。

### 第二步：只开 PWM，不启动闭环

- 保持输出占空比在中点附近，确认死区、互补输出、驱动波形是否正常。
- 重点确认 `eTMR0_CH1` 的 ADC 触发位置是否真的落在期望采样窗口。
- 先确认没有桥臂直通、电流毛刺、ADC 过载。

### 第三步：做电流极性检查

- 小电流强拖或定向注入时，观察 `Ia/Ib` 符号是否符合预期。
- 若符号反了，优先处理采样极性，不要先硬调 PI 或 SMO。

### 第四步：启动强拖，不急着交给观测器

- 先把 `align_current_a` 和 `drag_current_a` 设小。
- 让电机能稳定定向、低速拉起来即可。
- 如果有明显抖动或反向拉扯，先查相序和电流极性。

### 第五步：再调 SMO 交接

- 确认强拖阶段机械旋转平稳后，再看观测器 `theta/speed` 是否跟得上。
- `slide_gain` 先从偏小开始，逐步增大。
- `valid_bemf_v` 不要设太小，否则低速噪声会误判为有效反电势。
- `handover_electrical_hz` 建议先保守，确保反电势足够清晰后再提前交接。

### 第六步：最后再提带宽和速度

- 电流环先稳定，再提速度环。
- 电流环带宽建议从 `1000 Hz` 起步，稳定后再尝试 `1200~2000 Hz`。
- 每次只改一个参数组，不要同时改 PI、SMO、启动电流、交接速度。

## 7. 实机调试时最常见的问题

- 一启动就冲电流：
  - 先查电流采样极性、相序、零点偏置。
- 能转但抖动大：
  - 先降强拖电流和交接速度，再调 SMO 增益。
- 强拖能转，切观测器就丢步：
  - 说明交接太早，或者 SMO 参数还没收敛。
- VBus 显示不准：
  - 优先检查分压电阻值和 ADC 参考电压是否真的为 `5 V`。
- 温度异常跳变：
  - 当前软件还是线性温度模型，如果你板上实际是 NTC，需要后续换成 NTC 模型。
